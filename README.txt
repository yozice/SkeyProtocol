Курсовой проект на основе лабораторной работы 1.
Для первого входа в программу необходимо ввести имя учетной записи: Admin (пароль неважно)
Журнал действий пользователей находится в файле: log.txt
База данных пользователей находится в файле: data.txt

Реализация протокола основана на обмене сообщениями между 2 формами. 
LoginForm отправляет имя пользователя. В ответ получает номер одноразового ключа либо сообщение о несуществовании пользователя.
Если получен номер n, то пароль хешируется в LoginForm n-1 раз и отправляется в ActSelForm для проверки. 
В ActSelForm хешируем дополнительно один раз полученную фразу и сравниваем со значением в базе пользователей.
Если ключи совподают, тогда записываем в базу пользователей новое значение пароля и номера ключа(на 1 итерацию меньше).
